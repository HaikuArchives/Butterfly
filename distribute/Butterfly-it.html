Descrizione Visualizzatore d'immagini
Note Freeware

Butterfly è un visualizzatore d'immagini che adopera i traduttori per mostrare immagini bitmap. E' stato ispirato da Sea, un visualizzatore DOS di immagini. Il suo scopo è di permettere una navigazione facile e veloce di un considerevole numero di immagini.

C a r a t t e r i s t i c h e
  • Navigazione del File System
  • Supporto per il drag-and-drop (segue i collegamenti)
  • Ridimensiona dinamicamente le bitmap a riempire la finestra
  • Ingrandisce e diminuisce
  • Converte e salva in altri formati
  • Trascina usando il mouse
  • Supporto per lo sfondo del desktop
  • Supporto multilingua: inglese, francese, russo, italiano
U s o

Puoi usare il mouse o la tastiera per navigare.

Per aprire una directory, fai doppio click o premi Enter. Il percorso attuale è mostrato nella barra di stato.

Per visualizzare un file immagine, clicca sul suo nome o usa le frecce della tastiera per selezionarlo.

Il Drag'n'Drop è supportato in varie maniere. Puoi trasportare l'icona di un file immagine sull'icona di Butterfly per lanciarlo. Puoi trasportare l'icona di un file immagine o di una directory sulla finestra di Butterfly.

Fare click col tasto destro del mouse nella finestra dell'immagine permette di aprire un menu contestuale con varie opzioni (guarda sotto).

Comandi
+/- ingrandisci/diminuisci
* resetta il fattore di ingrandimento a 100%
[Spazio] o . attiva la vera modalità schermo intero (niente tranne l'immagine)
   
Freccia Su si muove su nella lista dei file
Freccia Giù si muove giù nella lista dei file
Backspace va alla directory superiore
Canc cancella il file selezionato
   
Shift + Freccia muove la foto nella direzione indicata
   
Tasto Sinistro Del Mouse trasporta l'immagine
Tasto Destro Del Mouse mostra il menu di popup
Menu Di Popup
Converti... traduce e salva l'immagine attuale in un'altro formato
Sfondo del desktop attuale imposta l'immagine come sfondo del desktop attuale
Sfondo di tutti i desktop imposta l'immagine come sfondo di tutti i desktop
Ingrandisci/Diminuisci indovina...

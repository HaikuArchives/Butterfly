Description Visionneur d'images
Note Freeware

Butterfly est un visionneur d'images basé sur les translators pour l'affichage des images. Il est inspiré de Sea, un visionneur d'images sous MS-DOS. Son objectif est de permettre une navigation et une visualisation facile et rapide d'un nombre important d'images.

U t i l i s a t i o n

La navigation dans les répertoires peut se faire en utilisant soit le clavier, soit la souris.

Pour rentrer dans un répertoire, double cliquez dessus ou pressez la touche entrée. Le chemin du répertoire courant est affiché dans la barre d'état.

Pour afficher une image, il suffit de la sélectionner un cliquant sur son nom dans la liste des fichiers ou d'utiliser les flèches du clavier pour se positionner dessus ou de rouler la roulette!

Le drag'n'drop est supporté de plusieurs facons. Vous pouvez déplacer l'icone d'une image sur l'icone du programme Butterfly pour le lancer dans le répertoire de l'image. Vous pouvez déplacer l'icone d'une image ou d'un répertoire dans la fenêtre de Butterfly déjà ouverte.

Cliquer avec le bouton droit de la souris sur l'image fait apparaitre un menu contextuel avec différentes actions et options possibles (voir plus bas).
